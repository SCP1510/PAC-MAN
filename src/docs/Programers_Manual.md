# Manual de Programador (Borrador)

## Arquitectura

El proyecto está organizado en los siguientes paquetes:

- model: contiene las entidades del juego
- logic: contiene la lógica del juego

## Flujo del juego

Main → Game → GameLoop → update/render

## Tecnologías
- Java
- JavaFX

## Estado actual
