# Manual de Usuario (Borrador)

## Descripción
Este proyecto es una implementación del juego Pac-Man utilizando JavaFX.

## Controles
- W / Flecha arriba → mover arriba
- S / Flecha abajo → mover abajo
- A / Flecha izquierda → mover izquierda
- D / Flecha derecha → mover derecha

## Estado actual
- Movimiento funcional
- Animación básica del personaje

## Instrucciones de uso
1. Ejecutar el archivo Main.java
2. Controlar a Pac-Man con el teclado
